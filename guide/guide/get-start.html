# 开始

# 环境准备

本地环境需要安装 pnpm在新窗口打开、Node.js在新窗口打开 和 Git在新窗口打开

注意

  • 使用pnpm在新窗口打开，否则依赖可能安装不上。
  • Node.js在新窗口打开 版本推荐 14.x 及以上。

# 工具配置

如果您使用的 IDE 是vscode在新窗口打开(推荐)的话，可以安装以下工具来提高开发效率及代码格式化

  • Iconify IntelliSense在新窗口打开 - Iconify 图标插件
  • windicss IntelliSense在新窗口打开 - windicss 提示插件
  • I18n-ally在新窗口打开 - i18n 插件
  • Volar在新窗口打开 - vue3 开发必备
  • ESLint在新窗口打开 - 脚本代码检查
  • Prettier在新窗口打开 - 代码格式化
  • Stylelint在新窗口打开 - css 格式化
  • DotENV在新窗口打开 - .env 文件 高亮

# 代码获取

注意

注意存放代码的目录及所有父级目录不能存在中文、韩文、日文以及空格，否则安装依赖后启动可能会出错。

# 从 GitHub 获取代码

# clone 代码
$ git clone https://github.com/toimc-team/vue3-toimc-admin.git

# 从 Gitee 获取代码

如果从 github clone 代码较慢的话，可以尝试用 Gitee在新窗口打开 同步代码到自己的仓库，再 clone 下来即可。

也可以通过下方地址进行 clone

$ git clone https://gitee.com/toimc-team/vue3-toimc-admin

# 安装

# 安装 Node.js

如果您电脑未安装Node.js在新窗口打开，请安装它。

验证

# 查看node版本
$ node -v
# 查看npm版本
$ npm -v

如果你需要同时存在多个 node 版本，可以使用 Nvm在新窗口打开 或者其他工具进行 Node.js 进行版本管理。

# 安装依赖

# Pnpm安装

必须使用 pnpm在新窗口打开进行依赖安装（若其他包管理器安装不了需要自行处理）。

如果未安装pnpm，可以用下面命令来进行全局安装

# 全局安装pnpm
$ npm install -g pnpm@next-7
# 验证 查看pnpm的版本
$ pnpm -v

# 依赖安装命令

在项目根目录下，打开命令窗口执行，耐心等待安装完成即可

$ pnpm install

    "lint:stylelint": "stylelint --cache --fix \"**/*.{vue,less,postcss,css,scss}\" --cache --cache-location node_modules/.cache/stylelint/",

  },
